## Printing in groovy

- `println()` adds a new line after the text is printed.

```groovy
println "Hello World"
println("Hello World")

print("hello world")
```

## Comments

```groovy
// Single line comment

/*
    Multi
    Line
    Comment
*/

/**
* GroovyDoc comment
* It is used for documentation
* @param a is the first number
```
